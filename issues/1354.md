#### <img src="https://avatars.githubusercontent.com/u/2818758?v=4" width="50">[koalaman](https://github.com/koalaman) commented at [2018-10-15T20:43:14Z](https://github.com/koalaman/shellcheck/issues/1354#issuecomment-430005995):

It sounds like you are using the Cabal build. This is the bleeding edge build that uses the latest available version of all packages and does not try to maintain compatibility further back than Debian Stable. Many of the dependencies no longer support GHC 7.6.3, and ShellCheck [blacklists](https://github.com/koalaman/shellcheck/blob/master/ShellCheck.cabal#L50-L51) it because of a GHC optimizer bug.

However, ShellCheck also has a Stack build which aims to solve exactly this long term compatibility problem. Try that instead.

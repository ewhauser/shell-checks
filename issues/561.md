# [\#561 Issue](https://github.com/koalaman/shellcheck/issues/561) `closed`: SC2079 triggers weirdly for associative array indices

#### <img src="https://avatars.githubusercontent.com/u/2818758?v=4" width="50">[koalaman](https://github.com/koalaman) opened issue at [2015-12-15T01:34:23Z](https://github.com/koalaman/shellcheck/issues/561):

This is another side effect of parsing associative array indices as numeric:

```
declare -A foo
foo[2015.12.14]=foo
```

SC2079 (( )) doesn't support decimals. Use bc or awk.

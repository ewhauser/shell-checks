# [\#2222 Issue](https://github.com/koalaman/shellcheck/issues/2222) `open`: Is there a built in way to apply a diff patch?

#### <img src="https://avatars.githubusercontent.com/u/51051860?u=bd52d7d729f496b2e8f4d8e4e02225e6b277da86&v=4" width="50">[dudicoco](https://github.com/dudicoco) opened issue at [2021-04-27 20:17](https://github.com/koalaman/shellcheck/issues/2222):

Hi,

I've read that order to auto-fix files with shellcheck, we can use the following options:
```
shellcheck file.sh -f diff | git apply
shellcheck file.sh -f diff | patch -p 1
```

When running shellcheck in the context of a pre-commit hook, it doesn't seem like it's possible to pipe the output in order to apply it.
The following doesn't work:
```
  - id: shellcheck
    language: docker_image
    entry: koalaman/shellcheck:v0.7.1 -f diff | patch -p 1
    types:
    - shell
```

I have also tried the https://github.com/shellcheck-py/shellcheck-py hook, but same problem.

Is there a way to apply the diff patch without resorting to pipe?

Thanks

#### <img src="https://avatars.githubusercontent.com/u/2818758?u=40d36e49d938fda80abd46d6347b9bace1f73dd4&v=4" width="50">[koalaman](https://github.com/koalaman) commented at [2021-05-01 18:00](https://github.com/koalaman/shellcheck/issues/2222#issuecomment-830670363):

There is no built in functionality for applying patches, no. I don't know pre-commit, but I imagine you can either have the alpine image do it:

    koalaman/shellcheck-alpine:v0.7.1 sh -c 'apk add patch && shellcheck -f diff "$@" | patch -p 1' _

or write a script that invokes `docker run ... | patch -p 1` and invoke it via a `system` hook

#### <img src="https://avatars.githubusercontent.com/u/51051860?u=bd52d7d729f496b2e8f4d8e4e02225e6b277da86&v=4" width="50">[dudicoco](https://github.com/dudicoco) commented at [2021-05-13 15:42](https://github.com/koalaman/shellcheck/issues/2222#issuecomment-840648212):

Thanks @koalaman.

I've ended up with the following solution:
```
- repo: https://github.com/shellcheck-py/shellcheck-py
  rev: v0.7.2.1
  hooks:
  - id: shellcheck
    name: Check sh files
    entry: bash -c 'shellcheck $@ -f diff | patch -p 1' --
```

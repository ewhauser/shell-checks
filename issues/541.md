# [\#541 Issue](https://github.com/koalaman/shellcheck/issues/541) `closed`: SC2094 and `sponge'

#### <img src="https://avatars.githubusercontent.com/u/3650039?v=4" width="50">[admirabilis](https://github.com/admirabilis) opened issue at [2015-11-06T01:56:32Z](https://github.com/koalaman/shellcheck/issues/541):

There is a package by Joey Hess which is becoming pretty popular recently called [`moreutils`](https://joeyh.name/code/moreutils/). One of its tools is called `sponge`, which allows for writing safely into a file read in the same pipeline, and could be ignored by SC2094:

```
NAME
   sponge - soak up standard input and write to a file

SYNOPSIS
   sed '...' file | grep '...' | sponge file

DESCRIPTION
   sponge  reads  standard  input and writes it out to the specified file.
   Unlike a shell redirect, sponge soaks up all its input  before  opening
   the  output file. This allows constructing pipelines that read from and
   write to the same file.
```

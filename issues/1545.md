If you run the latest git commit, the JSON output will contain fixes. They're not available in any stable release yet though.

Even if you do have them, the JSON fixes are in the form of raw indexed string replacement instructions that are not trivial to apply. The plan is to add a simple `diff` compatible output before the next release.

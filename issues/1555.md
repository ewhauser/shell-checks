#### <img src="https://avatars.githubusercontent.com/u/2818758?v=4" width="50">[koalaman](https://github.com/koalaman) commented at [2019-04-27T22:34:27Z](https://github.com/koalaman/shellcheck/issues/1555#issuecomment-487324851):

The docker image `koalaman/shellcheck` is intended to be run on a set of files similar to `shellcheck` itself:

    docker run -v "$PWD:/mnt" koalaman/shellcheck:stable myscript

It's not intended to run as a daemon or service. There is currently no docker image serving up shellcheck.net.

#### <img src="https://avatars.githubusercontent.com/u/464648?v=4" width="50">[austin987](https://github.com/austin987) commented at [2019-04-25T15:43:27Z](https://github.com/koalaman/shellcheck/issues/1557#issuecomment-486727749):

The problem is the backticks, not how date was invoked. `date` and $(date) will still show the date/time info, just maybe not in your preferred format.
```
austin@laptop:~$ echo "Current time: `date`"
Current time: Thu Apr 25 10:42:23 CDT 2019
austin@laptop:~$ echo "Current time: $(date)"
Current time: Thu Apr 25 10:42:28 CDT 2019
```

Your suggested code is even more broken:
```
austin@laptop:~$ date "+Current time: "
Current time: 
```

#### <img src="https://avatars.githubusercontent.com/u/1909753?u=1f25dc7199de6cfa4c7f3e95b5c942fad537aa18&v=4" width="50">[brackendawson](https://github.com/brackendawson) commented at [2023-03-09 18:32](https://github.com/koalaman/shellcheck/issues/2708#issuecomment-1462572201):

No, the check is correct, here's an example:

problematic:
```
nc -l 1337 &;
sleep 2;
nc -zv localhost 1337
```
correct:
```
nc -l 1337 &
sleep 2
nc -zv localhost 1337
```
Just remove the semicolons

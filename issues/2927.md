```bash

#!/bin/bash
x=something
if test -v x; then
        echo set
else
        echo unset
fi     
```

```
$ shellcheck myscript

x=something
^-- [SC2034](https://www.shellcheck.net/wiki/SC2034) (warning): x appears unused. Verify use (or export if used externally).
```

#### Here's what I wanted or expected to see:

no output. `x` is used/referenced. 


My use-case is following:

```bash
while [[ "$#" -gt 0 ]] ; do
    case "$1" in
	-n) readonly nflag=true; shift 1;;
        *) usage;;
    esac
done
if test -v nflag; then
    do_n_stuff
else
    do_non_n_stuff
fi
```

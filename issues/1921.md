How do I install shellcheck 0.7.0 version in Ubuntu 18.04. When I run **apt-get install shellcheck**. Currently it installs only 0.4.6 but vs code I need 0.7.0 as a prerequisite. 

#### <img src="https://avatars.githubusercontent.com/u/54114043?v=4" width="50">[dirk-seynhaeve](https://github.com/dirk-seynhaeve) commented at [2020-04-18 15:16](https://github.com/koalaman/shellcheck/issues/1921#issuecomment-615887937):

1. Go to the page with installation instructions:
https://github.com/koalaman/shellcheck#user-content-installing
2. Locate the section on the pre-compiled binaries for the latest release:
![image](https://user-images.githubusercontent.com/54114043/79641433-6af28e00-814c-11ea-98a9-b6c9afbb22b5.png)
3. Select the statically linked package for `x86_64`, to download the archive.
4. Extract the archive `tar xf shellcheck-stable.linux.x86_64.tar.xz`; This will give you an executable for `shellcheck` (I think we're on stable release 0.7.1 now).
5. Put this somewhere in your PATH directories, so you can pick it up (it is statically linked, a single executable with no library dependencies...

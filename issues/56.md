#### <img src="https://avatars.githubusercontent.com/u/302055?v=4" width="50">[deryni](https://github.com/deryni) commented at [2015-06-04T01:10:35Z](https://github.com/koalaman/shellcheck/issues/56#issuecomment-108668802):

You set the variable `a` to the value `=b` actually. Not `a=`.

`${a=}` is the unset-only version of the `:=` default value assignment expansion (i.e. it sets the value of the variable `a` to the empty string if `a` is unset at the time of that expansion.

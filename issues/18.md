# [\#18 Issue](https://github.com/koalaman/shellcheck/issues/18) `closed`: Questionable tip on replacing "$(dirname …)"

#### <img src="https://avatars.githubusercontent.com/u/1068245?v=4" width="50">[jhermann](https://github.com/jhermann) opened issue at [2013-10-28T11:00:37Z](https://github.com/koalaman/shellcheck/issues/18):

This is questionable, since it ignores the very real cornercase of "$0" being a plain word ("dirname foo" is ".", not "foo").

```
scriptdir="$(cd "$(dirname "$0")" && pwd)"
                   ^-- Use parameter expansion instead, such as ${var%/*}.
```

# [\#1526 Issue](https://github.com/koalaman/shellcheck/issues/1526) `closed`: Fail to install shellcheck on Ubuntu 14.04.5 Docker image

#### <img src="https://avatars.githubusercontent.com/u/234505?v=4" width="50">[matsev](https://github.com/matsev) opened issue at [2019-03-19T08:23:15Z](https://github.com/koalaman/shellcheck/issues/1526):

Installation of shellcheck on Ubuntu 14.04.05 Docker image using `apt-get` fails

----

### Step by step

1. Download and run Ubuntu 14.04.05 as a Docker container locally:
```sh
docker pull ubuntu:14.04.5
docker run -it --rm ubuntu:14.04.5
```

2. Attempt to install shellcheck in the running Docker container:
```sh
sudo apt-get update -y
[...]
sudo apt-get install shellcheck
```

### Expected result:
- shellcheck should be installed successfully

### Actual result:
```
Reading package lists... Done
Building dependency tree
Reading state information... Done
E: Unable to locate package shellcheck
```


----

### Background
We are using AWS CodeBuild as part of continuous integration system and it has been configured to use the [aws/codebuild/nodejs:10.14.1](https://docs.aws.amazon.com/codebuild/latest/userguide/build-env-ref-available.html) image, which in turn is based on Ubuntu 14.04.5 (see the [Dockerfile](https://github.com/aws/aws-codebuild-docker-images/tree/master/ubuntu/nodejs/10.14.1)). As part of the builds, we would like to install and use shellcheck to statically analyse the bash scripts in our projects.

#### <img src="https://avatars.githubusercontent.com/u/234505?v=4" width="50">[matsev](https://github.com/matsev) commented at [2019-07-08T21:47:11Z](https://github.com/koalaman/shellcheck/issues/1526#issuecomment-509404574):

Problem solved by updating to the [aws/codebuild/standard:2.0](https://docs.aws.amazon.com/codebuild/latest/userguide/build-env-ref-available.html) CodeBuild image (which is based on `ubuntu:18.04`, see the [Dockerfile](https://github.com/aws/aws-codebuild-docker-images/blob/master/ubuntu/standard/2.0/Dockerfile)). Consequently, we were able to install shellsheck successfully.

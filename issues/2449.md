#### Update the wiki page

Rationale: -maxdepth is not a valid option on AIX's `find` implementation.

```
# find . -maxdepth 1 -name "test*"
find: 0652-017 -maxdepth is not a valid option.

# uname
AIX

# which find
/usr/bin/find
```

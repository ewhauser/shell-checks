Encourage coders to specify a shebang line in each shell script. Another shell linter, [shlint](https://github.com/duggan/shlint), already does this.

Shebangs are good!
- For users, ensuring scripts are run with the intended shell, rather than the default system shell
- For text editors, helping to load syntax highlighters
- For developers, a visual reminder of the language the script is written in.
- For shellcheck, to identify scannable (sh/bash/derivatives) and non-scannable (csh, tcsh, fish) files.

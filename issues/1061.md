#### <img src="https://avatars.githubusercontent.com/u/1780970?v=4" width="50">[SerCeMan](https://github.com/SerCeMan) commented at [2017-12-30T05:10:55Z](https://github.com/koalaman/shellcheck/issues/1061#issuecomment-354528119):

@mordant23, @LukeShu wrote above one of the typical patterns that can be used to solve this problem.

Shellcheck can be run with:
```bash
shellcheck --plugins=myplugin.bin
```

Execution:

* `shellcheck` calls myplugin.bin and dumps AST to stdout
* `myplugin.bin` parses AST from stdin (it might use the same data structures as shellcheck uses, e.g. include shellcheck as a library)
* `myplugin.bin` dumps a set of warning to stdout
* `shellcheck` includes them in the final output

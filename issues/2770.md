#### Here's a snippet or screenshot that shows the problem:

```sh
#!/bin/bash

set -Eeuo pipefail

function get_x() {
  local -r x=$(cmd-that-fails "$1" | cmd-that-fails-moar)
  echo "$x"
}

x=$(get_x "abc")
```

#### Here's what shellcheck currently says:

```console
$ shellcheck myscript
No issues detected!

$
```

#### Here's what I wanted or expected to see:

```console
$ shellcheck myscript
 
Line 6-7:
  echo "$x"
^-- [SC2155](https://www.shellcheck.net/wiki/SC2155) (warning): Assigning a command output to $x and then returning the result via `echo $x` will not propagate a potential exit code. Replace it with the command instead.

$
```

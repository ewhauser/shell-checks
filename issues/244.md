# [\#244 Issue](https://github.com/koalaman/shellcheck/issues/244) `open`: Enhancement: Undefined functions

#### <img src="https://avatars.githubusercontent.com/u/166207?v=4" width="50">[mietek](https://github.com/mietek) opened issue at [2014-10-23T11:54:36Z](https://github.com/koalaman/shellcheck/issues/244):

Consider a _bash_ project consisting of multiple files, each declaring a number of functions.

It would be great if ShellCheck could flag any use of undeclared functions, as these are likely to be typos.

For bonus points, add some form of type checking—perhaps based on [success typings](http://user.it.uu.se/~kostis/Papers/succ_types.pdf), like the Erlang Dialyzer.

# [\#198 Issue](https://github.com/koalaman/shellcheck/issues/198) `open`: Need to quote parameter expansions inside of parameter expansions

```
phrase="this is a ?([[:space:]])phrase"; phrase=${phrase% ${phrase##* }}; echo "$phrase"
```

vs

```
phrase="this is a ?([[:space:]])phrase"; phrase=${phrase% "${phrase##* }"}; echo "$phrase"
```

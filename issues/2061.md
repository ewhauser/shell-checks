#### Here's a snippet or screenshot that shows the problem:

```sh
#!/bin/bash

# Remove all shell functions
function killfuncs() {
	BASHFCNS=$(declare -F | cut -f3 -d" ")
	unset "$(BASHFCNS)"
}
```

```
$ 
```

ShellCheck is right, this variable is unused because you are calling it as a function instead of expanding it as a variable. What you wanted was `unset $BASHFCNS`

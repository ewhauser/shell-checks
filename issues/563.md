#### <img src="https://avatars.githubusercontent.com/u/302055?v=4" width="50">[deryni](https://github.com/deryni) opened issue at [2015-12-16T17:53:21Z](https://github.com/koalaman/shellcheck/issues/563):

This is probably related to #108 (and I feel like there was another heredoc related issue that I saw somewhat recently but couldn't find it now) but this snippet

```
#!/bin/sh
false <<EOF || { ret=$?; exit $ret; }
EOF
exit $ret
```

Reports [SC2154](https://github.com/koalaman/shellcheck/wiki/SC2154) on the usage of `$ret` in the `exit` command on line 2 but does _not_ report it on line 4.

Probably relatedly, shellcheck does _not_ report [SC2086](https://github.com/koalaman/shellcheck/wiki/SC2086) on line 2 but _does_ report it on line 4.

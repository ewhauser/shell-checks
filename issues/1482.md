Dear,

calling the special variable `$!` trigger false positive error.
This special variable is not bound to function statement. Indeed this variable return last pid.

Minimal case:

```bash
#!/usr/bin/env bash

get_pid(){
  ls ~/ &
  echo "$!"
}

bug_sc2119(){
  pid=$( get_pid )
}
```

Thanks for your works

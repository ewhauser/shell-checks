This is the same as https://github.com/koalaman/shellcheck/issues/1161 just for Podman.

I am running one container with docker and one with podman. The same command-line gives an error when using podman.

This gives no SC2016:
```
docker run --read-only -d alpine /bin/sh -c 'i=0; while true; do echo $i; i=$(expr $i + 1); sleep 1; done'
```
This gives a SC2016:
```
podman run --read-only -d alpine /bin/sh -c 'i=0; while true; do echo $i; i=$(expr $i + 1); sleep 1; done'
```

adding `,"podman" -- like above` to src/ShellCheck/Analytics.hs should fix this.

#### <img src="https://avatars.githubusercontent.com/u/2818758?v=4" width="50">[koalaman](https://github.com/koalaman) commented at [2018-10-22T01:06:48Z](https://github.com/koalaman/shellcheck/issues/1359#issuecomment-431720396):

ShellCheck is unable to give much useful information because ` |mail -s "New Account for $user" -b root $user` is a syntax error. To break a line around a `|`, do it *after* the `|` and not before it:

    echo ... |
      mail ...

ShellCheck will now warn you about this with a SC1133. GitHub is currently struggling with technical issues, but should start showing the wiki page and commit later.

Note that there are several other issues that will prevent the script from working correctly. This issue tracker is purely about what shellcheck does or doesn't report, and not about fixing scripts in general, so try a site like StackOverflow instead.

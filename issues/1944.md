#### Here's a snippet or screenshot that shows the problem:

```sh
#!/bin/bash

readonly setup_file=/path/to/setup.conf

if [ -e $setup_file ]
then
  source $setup_file
else
  echo does not exit
fi
  
```

#### Here's what shellcheck currently says:
```
Line 7	SC1090: Can't follow non-constant source. Use a directive to specify location.
```
#### Here's what I wanted or expected to see:

since "setup_file" is a read-only variable, `source $setup_file` should be the treated the same as `source /path/to/setup.conf`.

# [\#2175 Issue](https://github.com/koalaman/shellcheck/issues/2175) `open`: Shellcheck fails to notice invalid interpreter where it could

#### <img src="https://avatars.githubusercontent.com/u/2000941?v=4" width="50">[RoelVdP](https://github.com/RoelVdP) opened issue at [2021-03-13 00:54](https://github.com/koalaman/shellcheck/issues/2175):

```
$ cat bug_demo.sh 
#!/bin/bash__
echo 'test'
$ ./bug_demo.sh 
bash: ./bug_demo.sh: /bin/bash__: bad interpreter: No such file or directory
$ shellcheck bug_demo.sh 
$ 
```
Proposed fix: do a  
```
if [ ! -r "/bin/bash__" -o ! -f "/bin/bash__" ]; then ...
```
(Or `whereis`) like check on the interpreter

#### The code:

#### <img src="https://avatars.githubusercontent.com/u/2818758?v=4" width="50">[koalaman](https://github.com/koalaman) commented at [2019-07-01T02:08:05Z](https://github.com/koalaman/shellcheck/issues/1589#issuecomment-507091273):

This is an interesting case because the quotes in `"STEAMUSER"` actually do serve a purpose, even if it's a strange one: they're positioned to split up the `$` and following characters so the `$` will be literal as well. This is more commonly done by escaping or using single quotes.

ShellCheck's intention was to suggest e.g. `eval steamu="\$STEAMUSER$i" where quotes are used for more canonical purposes.

544047c5afa adds [SC1135](https://github.com/koalaman/shellcheck/wiki/SC1135) to suggest using backslash instead of ending the quote, which makes the SC2140 more reasonable.

Thanks!

# [\#235 Issue](https://github.com/koalaman/shellcheck/issues/235) `closed`: False positive for SC2120 and SC2119

#### <img src="https://avatars.githubusercontent.com/u/243351?v=4" width="50">[brother](https://github.com/brother) opened issue at [2014-10-17T07:15:10Z](https://github.com/koalaman/shellcheck/issues/235):

``` shell
#!/bin/bash

options[0]="first"
options[1]="second"
options[2]="another one"

select_option () {
       while :; do
               echo
               echo "Select one of these:"
               for (( i=0 ; i < ${#options[*]}; i++ )); do
                       echo "$((i+1))) ${options[i]}"
               done | column
               echo
               read -p "Please specify the option to use: (a to abort) "
               if [[ $REPLY == [aA] ]]; then
                       echo -e "\nAborted\n" ; return 1
               elif [[ $REPLY =~ ^[0-9]+$ ]]; then
                       if (( REPLY > ${#options[*]} || REPLY == 0)); then
                               echo "Invalid option selection"
                       else
                               SELECTION=${options[$REPLY-1]%%:*}
                               return
                       fi
               else
                       echo "Unknown input"
               fi
       done
}

if select_option; then
    echo "You selected '$SELECTION'"
else
    echo "You skipped the selection"
fi
```

This will render the following errors:

```
brother /tmp/shellcheck$ shellcheck test.sh

   ^-- SC2119: Use select_option "$@" if function's $1 should mean script's $1.
```

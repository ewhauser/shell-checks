#### Here's what shellcheck currently says:

#### Here's what I wanted or expected to see:
the combination `&lt;` seems to be the html equivalent of `<` (less than) and is not translated in the html view, as this has been embedded in a code block. it should look like this (I assume):
`[[ $(echo "2 < 3.14" | bc) == 1 ]]  # Use bc`

If I am wrong, please excuse my error and discard this message.

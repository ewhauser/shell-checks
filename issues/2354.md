#### Here's a snippet or screenshot that shows the problem:

```sh
#!/bash
foofunc()
{
        echo I am foofunc
}
typeset -fx foofunc
```

#### Here's what shellcheck currently says:
```
Line 8	SC2034: foofunc appears unused. Verify use (or export if used externally).
```
#### Here's what I wanted or expected to see:

Nothing. From bash manpage, section typeset:
```
-f     Use function names only.
-x     Mark names for export to subsequent commands via the environment.
```

Because I am exporting them, they can be used elsewhere. Therefore, I contend that SC2034 (appears unused. Verify it or export it.) does not apply.

#### <img src="https://avatars.githubusercontent.com/u/6998367?v=4" width="50">[DomT4](https://github.com/DomT4) commented at [2017-01-09T05:36:39Z](https://github.com/koalaman/shellcheck/issues/821#issuecomment-271213969):

Your Homebrew is _way_ out of date. Try:

```bash
cd $(brew --repo) && git fetch && git reset --hard origin/master && brew update
```
And try your initial step again.

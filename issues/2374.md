because it was not working for me .. but i found why ..
I had some code (non-touched for long time) like this:

```shell
local -r IFS=$'\n'; platform=( "$(lscpu)" )

for line in ${platform[*]:-}; do
 :
 #...
done
```

i quoted the command to avoid SC2207 "Prefer mapfile or read -a to split command output (or quote to avoid splitting)."
This worked for me i guess because of IFS ... but in the moment i use quotes of course not ...

I changed this now like suggested in SC2207

```shell
    if [[ ${BASH_VERSINFO[0]:-} -ge 4 ]] ; then
        mapfile -t platform < <(lscpu)
    else
        platform=()
        while IFS='' read -r line; do platform+=("$line"); done < <(lscpu)
    fi

    local line
    for line in "${platform[@]:-}"; do
      :
     #...
   done
```

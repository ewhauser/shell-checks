#### <img src="https://avatars.githubusercontent.com/u/3129206?v=4" width="50">[matthewpersico](https://github.com/matthewpersico) commented at [2019-07-19T01:48:52Z](https://github.com/koalaman/shellcheck/issues/1646#issuecomment-513057720):

That's because it IS unused. From my bash command line:
```
$ declare -A sounds
$ sounds[bird]="tweet"
$ sounds[wolf]="howl"
$ echo sounds[bird]
sounds[bird]
$ echo ${sounds[bird]}
tweet
$ echo sounds[bird]=${sounds[bird]}
sounds[bird]=tweet
```
See the difference? You're asking to print a random string instead of the value.

There is no alternative written in https://www.shellcheck.net/wiki/SC2043 but I think it's pretty common to use run-once loops to help readability.

See https://github.com/qgis/QGIS/pull/54847#issuecomment-1748630050

#### <img src="https://avatars.githubusercontent.com/u/2818758?u=40d36e49d938fda80abd46d6347b9bace1f73dd4&v=4" width="50">[koalaman](https://github.com/koalaman) commented at [2023-12-11 04:23](https://github.com/koalaman/shellcheck/issues/2839#issuecomment-1849303493):

I edited the wiki to suggest ignoring if this is a stylistic choice. Thanks!

# [\#3200 Issue](https://github.com/koalaman/shellcheck/issues/3200) `open`: SC2157 should also check `test`

#### <img src="https://avatars.githubusercontent.com/u/13724985?v=4" width="50">[tgross35](https://github.com/tgross35) opened issue at [2025-05-04 05:59](https://github.com/koalaman/shellcheck/issues/3200):

```sh
#!/bin/sh

test -z SOMEARG  && echo y || echo n

[ -z SOMEARG ] && echo y || echo n
```

The `[ -z SOMEARG ]` line emits `SC2157 (error): Argument to -z is always false due to literal strings.`, as expected. However, the version with `test` does not get checked.

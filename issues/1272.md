#### <img src="https://avatars.githubusercontent.com/u/10137?v=4" width="50">[ghost](https://github.com/ghost) commented at [2018-08-25T18:45:57Z](https://github.com/koalaman/shellcheck/issues/1272#issuecomment-415988944):

I think that this ought to trigger SC2128. Consider the following sample.

```sh
#!/bin/bash
var=(foo bar baz)
[[ $var == foo\ * ]]
```

Shellcheck reports on it as follows:-

```
[[ $var == foo\ * ]]
   ^-- SC2128: Expanding an array without an index only gives the first element.
```

The use of read -ra seems to trip shellcheck up.

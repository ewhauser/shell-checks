Problem:
The tips in the [Recursiveness](https://github.com/koalaman/shellcheck/wiki/Recursiveness) wiki page how to handle recursive usage of the `shellcheck` command violates the [SC2044](https://github.com/koalaman/shellcheck/wiki/SC2044) rule.

Proposed solution:
Rewrite the  [Recursiveness](https://github.com/koalaman/shellcheck/wiki/Recursiveness) page so that it obeys  [SC2044](https://github.com/koalaman/shellcheck/wiki/SC2044).

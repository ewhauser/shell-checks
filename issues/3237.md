It would be great if I could ignore certain rules for an entire stage with something like `# hadolint stage ignore`. Example:

```dockerfile
# hadolint stage ignore=DL3004
FROM somebaseimage AS builder

RUN sudo touch /etc/someconfig.txt

FROM alpine

COPY --from=builder /etc/someconfig.txt /etc/someconfig.txt
```

It would be safer than using `# hadolint global ignore`.

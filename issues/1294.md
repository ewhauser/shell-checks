# [\#1294 Issue](https://github.com/koalaman/shellcheck/issues/1294) `closed`: Add support for printf format string for date/times

Bash' printf supports a special format string for date/time formatting like this:

printf -v CURRENT_DATE "%(%F %H:%M:%S)T"

At the moment, you get the following spurious warning for that kind of placeholder:

warning: This format string has 1 variables, but is passed 0 arguments. [SC2183]

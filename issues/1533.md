#### Here's a snippet or screenshot that shows the problem:

```bash
#!/usr/bin/env sh
x=1

# ShellCheck incorrectly warns here:
echo $(( ${x+1} ))  # ⇒ 1

echo $(( x+1 ))     # ⇒ 2
```

# [\#644 Issue](https://github.com/koalaman/shellcheck/issues/644) `closed`: shellcheck: In dash, LINENO is not supported. [SC2169]

#### <img src="https://avatars.githubusercontent.com/u/521538?v=4" width="50">[Earnestly](https://github.com/Earnestly) opened issue at [2016-04-06T20:46:49Z](https://github.com/koalaman/shellcheck/issues/644):

Initially added in [0df9679](http://git.kernel.org/cgit/utils/dash/dash.git/commit/?id=0df96793ef6aa103df228d7dfe56099b7d721a15) and improved in [5bb39bb](http://git.kernel.org/cgit/utils/dash/dash.git/commit/?id=5bb39bb1995cb12d8da76b1d482df9be1acc2eb4); dash has supported `$LINENO` for roughly six years.

``` sh
#!/bin/dash
# dash 0.5.8
printf '%d\n' "$LINENO"
```

```
% ./test.dash
3
```

PS: POSIX 2008 also appears to require `$LINENO` http://pubs.opengroup.org/onlinepubs/9699919799/

# [\#78 Issue](https://github.com/koalaman/shellcheck/issues/78) `closed`: shellcheck does not report stray break in if statements

#### <img src="https://avatars.githubusercontent.com/u/243351?v=4" width="50">[brother](https://github.com/brother) opened issue at [2014-01-28T15:44:36Z](https://github.com/koalaman/shellcheck/issues/78):

``` bash
#!/bin/bash

a="hello"
if [[ $a != "bye" ]]; then
    echo "say something"
    break
else
    echo "ok"
fi
```

brother /tmp$ shellcheck test.sh

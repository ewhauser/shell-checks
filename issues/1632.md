#### <img src="https://avatars.githubusercontent.com/u/2818758?v=4" width="50">[koalaman](https://github.com/koalaman) commented at [2019-07-01T03:11:29Z](https://github.com/koalaman/shellcheck/issues/1632#issuecomment-507100570):

I'm assuming you've decided that your distro's version (`yum install shellcheck`) isn't sufficient.

The Linux x86_64 binaries are statically linked and therefore have no external dependencies. You can download the latest release version [here](https://storage.googleapis.com/shellcheck/shellcheck-stable.linux.x86_64.tar.xz) or see the complete list of versions (including latest builds) in the [GCE storage bucket listing](https://shellcheck.storage.googleapis.com/index.html).

You can just unpack it and place the `shellcheck` binary somewhere in your `$PATH`.

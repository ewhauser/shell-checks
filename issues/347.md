# [\#347 Issue](https://github.com/koalaman/shellcheck/issues/347) `closed`: SC2120/SC2119 Don't Consider Local Functions

#### <img src="https://avatars.githubusercontent.com/u/1458615?v=4" width="50">[willbuckner](https://github.com/willbuckner) opened issue at [2015-04-11T03:09:28Z](https://github.com/koalaman/shellcheck/issues/347):

SC2120 and SC2119 don't consider local functions/inner functions. Not only that, but I have to add "disable" comments in two places! :(

``` bash
$ cat test.sh
#!/bin/bash

function outer_function()
{
    function inner_function()
    {
        echo "Inner: $1"
    }

    inner_function "Foo."
    inner_function "Bar."

    unset -f inner_function
}

outer_function

$ bash test.sh
Inner: Foo.
Inner: Bar.
$ shellcheck test.sh

In test.sh line 3:
function outer_function()
^-- SC2120: outer_function references arguments, but none are ever passed.


In test.sh line 14:
outer_function
^-- SC2119: Use outer_function "$@" if function's $1 should mean script's $1.

$
```

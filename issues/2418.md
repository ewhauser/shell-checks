#### <img src="https://avatars.githubusercontent.com/u/2818758?u=40d36e49d938fda80abd46d6347b9bace1f73dd4&v=4" width="50">[koalaman](https://github.com/koalaman) commented at [2021-12-21 22:34](https://github.com/koalaman/shellcheck/issues/2418#issuecomment-999139481):

This was a problem with ShellCheck trying to match `\xNN` but running out of letters. This has now been fixed (along with multiple other correctness improvements like `\x1z`).  Affected versions can write out `$'\x01'` to avoid a crash.

Great catch, thanks!

#### <img src="https://avatars.githubusercontent.com/u/197253?v=4" width="50">[kurahaupo](https://github.com/kurahaupo) commented at [2025-03-12 04:00](https://github.com/koalaman/shellcheck/issues/2935#issuecomment-2716366004):

Does ShellCheck actually check any of the syntax between `[[` and `]]`?
If not, then `]` and `[` are just ordinary parameters inside one `[[`...`]]` command.

The following code is correct:
```
$ if [[ [ = [ ]] ; then echo Yes ; else echo No ; fi
Yes
$ if [[ ] = [ ]] ; then echo Yes ; else echo No ; fi
No
```

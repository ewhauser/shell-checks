#### <img src="https://avatars.githubusercontent.com/u/2818758?u=40d36e49d938fda80abd46d6347b9bace1f73dd4&v=4" width="50">[koalaman](https://github.com/koalaman) commented at [2022-04-22 03:45](https://github.com/koalaman/shellcheck/issues/2450#issuecomment-1105973355):

The error is certainly misleading, but this construct does require quoting:

```
$ mkdir -p https:/2/foo
$ echo https://[2620:10d:c0a8:2b::127]/
https://2/
```

While a directory named `https:` is unlikely, it will also trigger `nullglob` and `failglob` so it's better to quote it. I've updated the wiki. Thanks!

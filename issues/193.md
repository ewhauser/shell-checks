```
regex="~(E)^This\sis\the\sline$"

do
  :
done
{fd}<&-
```
I'm hitting SC1083 and hoping to not need an eval workaround to get past the SDLC police.  It's freakishly fast, so I also want to make sure others know about this feature, and I'm hopeful it makes it's way into bash.

Hi, Thank you for making awesome software!

Shellcheck has no such an option, right?

I would like to add Shellcheck syntax checker for [flycheck](http://flycheck.readthedocs.org/en/latest/) which is syntax checking extension for Emacs, and I needed such an option for the following reasons:
- Emacs'shell mode has shell guessing mechanism, and I want to check shellscript according to guessed shell.
- No matter how there is no shebang , I want to check for specific shell.

In connection with: https://github.com/flycheck/flycheck/issues/267

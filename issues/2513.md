#### <img src="https://avatars.githubusercontent.com/u/1323808?u=03604720cc3b20815ec7c7c300b020dfd28b4f55&v=4" width="50">[danielhoherd](https://github.com/danielhoherd) commented at [2022-06-08 18:27](https://github.com/koalaman/shellcheck/issues/2513#issuecomment-1150251247):

~There is already an arm64 native build.~ Scratch that. Homebrew builds it. They have a stored binary, but `brew` authenticates when it downloads the binary, so it's not generally available outside the `brew` cli.

```
$ file /opt/homebrew/Cellar/shellcheck/0.8.0/bin/shellcheck
/opt/homebrew/Cellar/shellcheck/0.8.0/bin/shellcheck: Mach-O 64-bit executable arm64
$ lipo -archs /opt/homebrew/Cellar/shellcheck/0.8.0/bin/shellcheck
arm64
```

#### <img src="https://avatars.githubusercontent.com/u/2818758?v=4" width="50">[koalaman](https://github.com/koalaman) commented at [2018-01-15T21:19:43Z](https://github.com/koalaman/shellcheck/issues/1094#issuecomment-357791491):

`shellcheck-alpine` was just changed to behave more like its `alpine` base image for #1069 (it's now just like `alpine:latest` but with shellcheck already installed). 

You can either have the alpine image run `shellcheck`:

    docker run -v `pwd`:/scripts koalaman/shellcheck-alpine shellcheck /scripts/scripts/clustertester.sh

or use the non-alpine image which has shellcheck as its entrypoint:

    docker run -v `pwd`:/scripts koalaman/shellcheck /scripts/scripts/clustertester.sh

If you're just running shellcheck and not extending the image for some specific purpose, the non-alpine image is smaller and better.

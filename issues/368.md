# [\#368 Issue](https://github.com/koalaman/shellcheck/issues/368) `closed`: SC2124 false positive

```
array=(a b c)
string=${array[@]: -1}
       ^-- SC2124: Assigning an array to a string! Assign as array, or use * instead of @ to concatenate.
```

`@` and `*` are equivalent in this case.

Maybe it's appropriate to add this to the exceptions in the wiki?

```sh

#!/bin/bash

a=( x y z )

# ShellCheck disable=SC2068

```


In "shellcheck disable" statements, the string

    # shellcheck disable=SC####

is recognized, but the string

    # Shellcheck disable=SC####

is not. I expected ShellCheck to recognize capital letters in disable statements.

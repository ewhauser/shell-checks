Sounds like you are using the vscode plugin for shellcheck. You probably need to make sure that works according to spec, IIRC you need to have shellcheck locally installed and there are multiple ways to do that depending on your platform - see https://github.com/koalaman/shellcheck/blob/master/README.md#installing. If you can execute `shellcheck --version` in a terminal on your system that part is working properly and the error is probably somewhere between that and the vscode plugin, that's a separate project: https://github.com/vscode-shellcheck/vscode-shellcheck/issues.

Shellcheck is not for scanning python code. Maybe checkout some other projects if your target is python: https://code.visualstudio.com/docs/python/linting.

Awesome tool. I ran into a few problems following the installation instructions. I didn’t want to just send a PR for the readme, as I’m not entirely sure whether my issue was just an issue with my system.

I found I had to run the following (separately and before installing shellcheck):

```
brew install cabal-install
cabal update
cabal install cabal-install
cabal install syb
```

I wondered whether the reason it was recommended to run `cabal install cabal-install` was because I neglected to run brew update first. Also, it didn’t feel right that syb failed to install as a dependency, but succeeded when specifically targetted. I failed to grab the whole log but you can see my output in my write up [here](http://markcrossfield.co.uk/2014-04-07-quality-shell-scripting-static-analysis.html#syb).

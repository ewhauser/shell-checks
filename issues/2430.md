#### <img src="https://avatars.githubusercontent.com/u/2818758?u=40d36e49d938fda80abd46d6347b9bace1f73dd4&v=4" width="50">[koalaman](https://github.com/koalaman) commented at [2022-01-17 21:39](https://github.com/koalaman/shellcheck/issues/2430#issuecomment-1014904685):

When I try your unquoted version, I get `Error: either "to" is duplicate, or "default" is a garbage.` because the host has two default routes (with different weights).

You could write it as

```
ip route list 0/0 |
  while read -ra params
  do
    ip route del "${params[@]}"
  done
```

Which would both avoid any warnings, and also correctly handle the cases when it outputs 0 or 2 routes.

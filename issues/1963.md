You installed it via apt and now want to know where the files are?
In the terminal type
`dpkg -L shellcheck`

Muting a specific rule can be done via a directive in the source code. Via the command line. Or via a config file. All of this is documented in the wiki. Start at https://github.com/koalaman/shellcheck/wiki/Ignore

#### <img src="https://avatars.githubusercontent.com/u/243351?v=4" width="50">[brother](https://github.com/brother) commented at [2020-05-27 18:49](https://github.com/koalaman/shellcheck/issues/1963#issuecomment-634868671):

At this current point in time you need backports for debian stable to have the shellcheckrc support - minimum shellcheck version for that is 0.7.0 - pure stable debian only offers 0.5.0. See table below.

```
    old-old-stable: 0.3.4
    oldstable: 0.4.4
    stable: 0.5.0
    stable backports: 0.7.1
    testing: 0.7.1
    unstable: 0.7.1
```

```sh

kubectl completion bash | shellcheck --shell bash -

```

In the online shellcheck.net tool, pasting the completion script gives the message that shellcheck exceeded the resources of its sandbox. shellcheck 0.7.1 on my machine hangs indefinitely and uses 100% cpu.

kubectl binary can be downloaded to test with

```sh
curl -LO "https://dl.k8s.io/release/$(curl -L -s https://dl.k8s.io/release/stable.txt)/bin/linux/amd64/kubectl"
```

[instructions here](https://kubernetes.io/docs/tasks/tools/install-kubectl/)

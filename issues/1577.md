✏️ @koalaman, in #882, you wrote:
> With [301705e][hash] you can specify `-P SCRIPTDIR` to search relative to the script's directory, or `-P /my/path` to search relative to a path in the filesystem.
> 
> This affects both relative and absolute paths, both in `source` directives and literal pathnames handed to `source`/`.`

### Are any of these supported options?

#### ➡️ `source` directive:

```bash
$ ls ${LIBDIR}
goodstuff.bash tryinghard.sh
$ cat ${LIBDIR}/tryinghard.sh
#!/bin/bash
# shellcheck source=SCRIPTDIR
source ${LIBDIR}/goodstuff.bash
```

#### 🔀 `source-path` directive:

```bash
#!/bin/bash
# shellcheck source-path=SCRIPTDIR
source ${LIBDIR}/goodstuff.bash
```

#### ⚙️ `.shellcheckrc` option

```console
$ cat .shellcheckrc
source-path=SCRIPTDIR:./libs
$
```

#### <img src="https://avatars.githubusercontent.com/u/3129206?v=4" width="50">[matthewpersico](https://github.com/matthewpersico) commented at [2019-10-25T00:18:45Z](https://github.com/koalaman/shellcheck/issues/1729#issuecomment-546150625):

ALLCAPS variables are assumed to be defined elsewhere (and possibles exported), so shellcheck does not complain about no previous encounter. Everything else is assumed to be script local, so shellcheck will throw 2154 for the second one.

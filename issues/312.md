#### <img src="https://avatars.githubusercontent.com/u/2818758?v=4" width="50">[koalaman](https://github.com/koalaman) commented at [2015-02-17T17:16:16Z](https://github.com/koalaman/shellcheck/issues/312#issuecomment-74707884):

Your example is actually one where `grep -c` does not return an integer:

```
$ test $(grep -c STRING file) -gt 5
grep: file: No such file or directory
-bash: test: -gt: unary operator expected
```

Since it doesn't output anything, quoting is required to preserve the empty string.

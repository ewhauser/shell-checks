**shellcheck.net Feature Request - Dark Mode option**

I have probably referenced [shellcheck.net](https://www.shellcheck.net) 500 or more times helping those new to shell scripting find this resource on SO and similar sites. I use it as well. One issue that really make it a bit painful to use (especially for older eyes) is when coming from working within a dark desktop theme, and other dark theme pages in a browser, is to hit the acres of blinding white background of shellcheck.net.

I know it would take a cookie or local storage to set the preference, but that is one cookie or bit of persistent local storage I would gladly accept.

It doesn't have to be anything fancy, the current interface is fine, it just need to have a preference to let the user select *light* or *dark* mode. I would even go so far as to suggest making the dark mode the default as it seems (thankfully) most sites are going in that direction.

Shellcheck.net is an incredible tool, works great, so no bug report, just a feature request from the older-eyes crowd.

In 74ea5ea, "else if" is parsed like "elif" with a warning.

This does mean that scripts that use "else if" correctly, like `if true; then true; else if true; then true; fi; fi;`, will fail to parse. In a realistic script with nested if statements, there will hopefully be a line feed after the "else".

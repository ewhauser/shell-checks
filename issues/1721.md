But consider how many other template systems are out there. Is `shellcheck` supposed to recognize them all, with their various set-off syntaxes? And, how would this work anyway? If bash DNE, how do lines 1 and 5 get executed? Perhaps a more real world example could help here? Probably easier here to not put real shell code in this, but extract said code to external script and just run that:
```
<% if File.exist?('/bin/bash') %>
/some/bin/say_bash_exists
<% else %>
<% rubyerror "bash is missing" %. ## hypothesizing, but if bash dne, you cannot 'echo', or assume any other shell that can, no?
<% end %>
```

```

<% end %>
```

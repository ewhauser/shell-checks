# [\#467 Issue](https://github.com/koalaman/shellcheck/issues/467) `closed`: additional bashism: printf %q

Using `%q` in `printf` is a famous bashism that is useful for creating reusable command line inputs, and even advanced shell scripters may unconsciously use it in `#!/bin/sh`.

#### Here's a snippet or screenshot that shows the problem:

```sh
#!/bin/sh

foo="--help"
case "$foo" in
    -h|--help)
      echo "usage: $0 [options]"
      exit 0
      ;;

    -*|--*)
      echo "Unknown option: $1"
      exit 1
      ;;

    *)
      echo "arg: $1"
      shift
      ;;
esac
```

#### Here's what shellcheck currently says:
```
In foo.sh line 10:
    -*|--*)
    ^-- SC2221 (warning): This pattern always overrides a later one on line 10.
       ^-^ SC2222 (warning): This pattern never matches because of a previous pattern on line 10.
```

No warning.

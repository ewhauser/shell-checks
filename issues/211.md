# [\#211 Issue](https://github.com/koalaman/shellcheck/issues/211) `closed`: SC2143 Needs a contradiction

#### <img src="https://avatars.githubusercontent.com/u/1583?v=4" width="50">[mikehale](https://github.com/mikehale) opened issue at [2014-08-13T16:59:43Z](https://github.com/koalaman/shellcheck/issues/211):

This recommendation: `SC2143 Instead of [ -z $(foo | grep bar) ], use ! foo | grep -q bar .` can result in an unexpected exit code of `141`. See: http://stackoverflow.com/questions/19120263/why-exit-code-141-with-grep-q

Example:

``` bash
$ seq 1 10000 | grep -q 1; echo $?
0
$ set -o pipefail
$ seq 1 10000 | grep -q 1; echo $?
141
```

# [\#721 Issue](https://github.com/koalaman/shellcheck/issues/721) `closed`: Create packages for Ubuntu 12.04 LTS and/or 14.04 LTS 

#### <img src="https://avatars.githubusercontent.com/u/48132?v=4" width="50">[elyezer](https://github.com/elyezer) opened issue at [2016-08-02T19:50:41Z](https://github.com/koalaman/shellcheck/issues/721):

It would be great to have shellcheck built for Ubuntu 12.04 LTS and/or 14.04 LTS because we can easily install it on Travis-CI environment and make shellcheck run for every PR sent to a repository.

If the package is available then the `.travis.yml` file would be like the following:

```
sudo: required
dist: trusty
before_install:
    - sudo apt-get -qq update
    - sudo apt-get install -y shellcheck
script:
    make lint
```

Instead of needing to compile from source. In this case, the `.travis.yml` will look like the following:

```
sudo: required
dist: trusty
before_install:
    - sudo apt-get -qq update
    - sudo apt-get install -y cabal-install
    - cabal update
    - git clone https://github.com/koalaman/shellcheck.git
    - cd shellcheck
    - cabal install
    - cd ..
    - export PATH="$HOME/.cabal/bin:$PATH"
    - which shellcheck
script:
    shellcheck ...
```

Also the advantage of having the package is that the build executing time is much faster.

> In the above case we can also reduce external program calls because there equals

echo isn't an external program call. But if you are worried about the number
of commands, the simplest version would be:

```
echo 1>&2 "\
*** Unable to find the ncurses libraries or the
*** required header files.
*** 'make menuconfig' requires the ncurses libraries.
***
*** Install ncurses (ncurses-devel) and try again.
***"
```

Also please don't recommend `echo -e` as it's unportable and broken in some
subtle ways; have a look at printf instead.

# [\#3086 Issue](https://github.com/koalaman/shellcheck/issues/3086) `closed`: Ignoring a wrapped command

#### <img src="https://avatars.githubusercontent.com/u/67508272?u=036bf159a21a5249e27da1315e46d9cdd3343222&v=4" width="50">[avidseeker](https://github.com/avidseeker) opened issue at [2024-11-15 23:34](https://github.com/koalaman/shellcheck/issues/3086):

Consider the example:

```sh
# shellcheck ignore=2046
xrandr --verbose \
	$(for d in DP-1 DP-2 DP-3 HDMI-1 HDMI1; do
		echo --output "$d" --off
	done)
```

What I tried, as per https://www.shellcheck.net/wiki/SC1143 suggestions
```sh
xrandr --verbose \
	`# shellcheck ignore=2046` \
	$(for d in DP-1 DP-2 DP-3 HDMI-1 HDMI1; do
		echo --output "$d" --off
	done)
```

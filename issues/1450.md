#### <img src="https://avatars.githubusercontent.com/u/2818758?v=4" width="50">[koalaman](https://github.com/koalaman) commented at [2019-01-11T02:55:22Z](https://github.com/koalaman/shellcheck/issues/1450#issuecomment-453356686):

No, there is currently no cross-platform install script. 

In general, you can easily install ShellCheck through the OS or distro's package mangers, such as `brew install shellcheck` on macOS and `apt install shellcheck` (or `snap install shellcheck`) on Ubuntu.

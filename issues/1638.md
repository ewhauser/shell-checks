#### Here's what I wanted or expected to see:

##### Using `%f` with built-in `printf` command

When formatting floating-point numbers with the built-in `printf`command, the floating-point number argument has to be represented according to the environment variable `LC_NUMERIC` rather than in the POSIX format.
- Either use `LC_NUMERIC=POSIX printf '%f' <floating-point-number-argument>` if both argument and print-out are to be using then POSIX floating-point numbers format,
- Either use `env printf '%f' <floating-point-number-argument>` to format the print-out according to the `LC_NUMERIC` environment variable, while still accepting POSIX formatted floating-point numbers arguments.

I am not sure if this is in the scope of this project but it would be nice if shellcheck could search the files to be sourced not in the current directory but where they are actually loaded from. It would be nice if for example ``cd`` calls would be taken under consideration. My use case would be editing scripts from different path than they are run. Or shellcheck path independent sources with something like this:

```
#!/bin/bash

CURRENTPATH="$(pwd)"
SCRIPTPATH="$( cd -- "$(dirname "$0")" || exit 1 >/dev/null 2>&1 ; pwd -P )"
cd "$SCRIPTPATH" || exit 1

# shellcheck disable=SC1091
source lib/lib.sh

cd "$CURRENTPATH" || exit 1
```

# [\#1317 Issue](https://github.com/koalaman/shellcheck/issues/1317) `open`: Latest apt repository version is 0.4.6

#### <img src="https://avatars.githubusercontent.com/u/10715719?v=4" width="50">[Tahler](https://github.com/Tahler) opened issue at [2018-08-13T23:54:10Z](https://github.com/koalaman/shellcheck/issues/1317):

Installing ShellCheck through `apt-get` installs 0.4.6, whereas installing with Cabal or on [Arch](https://www.archlinux.org/packages/community/x86_64/shellcheck/) installs 5.0.0.

```sh
$ apt-get update
$ apt-get install -y shellcheck
$ shellcheck -V
ShellCheck - shell script analysis tool
version: 0.4.6
license: GNU General Public License, version 3
website: http://www.shellcheck.net
```

#### <img src="https://avatars.githubusercontent.com/u/4347038?v=4" width="50">[virgilwashere](https://github.com/virgilwashere) commented at [2019-05-04T06:18:08Z](https://github.com/koalaman/shellcheck/issues/1317#issuecomment-489298631):

As I found out in #1566, there are pre-compiled builds available based on the last commit.

Quoting [README.md](README.md)
> you can download pre-compiled binaries for the latest release here:
>
>* [Linux, x86_64](https://storage.googleapis.com/shellcheck/shellcheck-stable.linux.x86_64.tar.xz) (statically linked)
>
> see the [storage bucket listing](https://shellcheck.storage.googleapis.com/index.html) for checksums

And create a current man page with:
```bash
pandoc -s -f markdown-smart -t man shellcheck.1.md -o shellcheck.1
```

`-smart` disables `--` as `en-dashes`.

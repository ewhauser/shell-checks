#### <img src="https://avatars.githubusercontent.com/u/2818758?u=40d36e49d938fda80abd46d6347b9bace1f73dd4&v=4" width="50">[koalaman](https://github.com/koalaman) commented at [2022-07-20 15:20](https://github.com/koalaman/shellcheck/issues/2539#issuecomment-1190421034):

ShellCheck actually requires the sourced file to give correct warnings about the *current* file. Consider something like:

```
#!/bin/bash
verbose=1
source abc
```

Should it trigger an unused assignment warning? It depends on whether `abc` uses it or not, and ShellCheck can only determine this by looking at it.

Syntax highlighting differs for strings spanning multiple lines, e.g. shell builtin are highlighted inside multi-line strings.
1. Go to http://www.shellcheck.net/.
2. Enter the following:
   
   ``` bash
   echo "and break break continue eval exec exit export if readonly return set"
   
   echo "
   and break break continue eval exec exit export if readonly return set"
   ```
3. Look at syntax highlighting.

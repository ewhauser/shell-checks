#### <img src="https://avatars.githubusercontent.com/u/22617577?v=4" width="50">[xpressos](https://github.com/xpressos) commented at [2018-10-31T01:26:50Z](https://github.com/koalaman/shellcheck/issues/1378#issuecomment-434528695):

**(Solved)**

1. Install Scoop (https://scoop.sh) on windows via powershell.
`iex (new-object net.webclient).downloadstring('https://get.scoop.sh')`

Note: if you get an error you might need to change the execution policy (i.e. enable Powershell) with Set-ExecutionPolicy RemoteSigned -scope CurrentUser
(must have netframework 4.5 or higher and Powershell 3 installed)

2.Open Powershell 3 and run install.
`scoop install shellcheck`

3.Install on SublimeText / Atom per instructions.
https://github.com/koalaman/shellcheck

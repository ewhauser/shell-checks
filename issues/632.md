#### <img src="https://avatars.githubusercontent.com/u/2818758?v=4" width="50">[koalaman](https://github.com/koalaman) commented at [2016-03-23T16:57:37Z](https://github.com/koalaman/shellcheck/issues/632#issuecomment-200440823):

You have no `.sh` files so xargs calls the program once with zero arguments.

Call shellcheck from a find `-exec` to both avoid calling it without input and to better handle special characters in filenames

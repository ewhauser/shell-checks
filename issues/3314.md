This function is like a syntax sugar for the if statement used in my repo

```sh
function require() {
    local check="$1"
    local opt="$2"
    # shellcheck disable=SC1009
    # shellcheck disable=SC1072
    if [ ! "$check" "$opt" ]; then
        echo "ERROR: Check [ ! "$check" "$opt" ] failed" 1>&2
        PROCESS_FAILED=2
        return 1
    fi
    return 0
}
```

Usage:
```sh
    require -f "system.tar.gz" || return
```

Shellcheck error:

SC1073 Couldn't parse this test expression. Fix to allow more checks.
SC1009
SC1072

# [\#99 Issue](https://github.com/koalaman/shellcheck/issues/99) `closed`: Treats string as decimal; [[ $var == "1.1" ]]

#### <img src="https://avatars.githubusercontent.com/u/1693206?v=4" width="50">[knirch](https://github.com/knirch) opened issue at [2014-02-19T13:48:20Z](https://github.com/koalaman/shellcheck/issues/99):

``` bash
var="foo"
if [[ $var == "1.1" ]]; then
    echo "yes"
fi
```

SC2072: Decimals are not supported. Either use integers only, or use bc or awk to compare.

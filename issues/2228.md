As documented here: https://github.com/CircleCI-Public/shellcheck-orb/issues/34. Shellcheck was firing and returning a 0 exits status, but not all files were matched and it was difficult to debug which files that shellcheck was processing.

I'm proposing adding a `--verbose` flag to the shellcheck CLI and I would like it to emit the names of files that it evaluates.

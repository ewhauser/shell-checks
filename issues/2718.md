#### Here's a snippet or screenshot that shows the problem:

```sh

#!/bin/bash
# shellcheck enable=check-extra-masked-returns
read -r a b v < <(echo this is something | grep nothing)
rc=$?       # return code of read
wait $!
rc2=$?     # return code of subprocess

```

#### Here's what shellcheck currently says:

SC2312 Consider invoking this command separately to avoid masking its return value (or use '|| true' to ignore).

#### Here's what I wanted or expected to see:

Actually the result of process substitution can be checked using wait and therefore the return value is not masked.
This solution does not work with the documented example: cd "$(command)/etc" you may say it is a different case.
Also it works only with bash 5.x (5.2 was tested), but does not work with bash 4.3.

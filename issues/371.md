# [\#371 Issue](https://github.com/koalaman/shellcheck/issues/371) `closed`: sc2034 false positive with declare -F

``` bash
#!/bin/bash

__custom_func() {
    echo "In Custom!"
}

declare -F __custom_func >/dev/null && __custom_func
```

```

                 ^––SC2034 __custom_func appears unused. Verify it or export it.
```

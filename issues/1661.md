#### <img src="https://avatars.githubusercontent.com/u/2818758?v=4" width="50">[koalaman](https://github.com/koalaman) commented at [2019-08-03T04:39:40Z](https://github.com/koalaman/shellcheck/issues/1661#issuecomment-517893787):

The statement `[[ "${ADB_INSTALL_OUTPUT}" =~ "Can't find service" ]]` is correct from Bash's point of view, and ShellCheck does not emit a warning. The quotes around `"Can't find service"` are not optional, as the single quote and spaces need to be interpreted literally (For quoted patterns that probably weren't intended to match literally, there's there's already [SC2076](https://github.com/koalaman/shellcheck/wiki/SC2076)).

ShellCheck is not involved in syntax highlighting, so if IntelliJ is coloring the string in an odd way, that's an issue with something else.

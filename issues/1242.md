#### <img src="https://avatars.githubusercontent.com/u/2818758?v=4" width="50">[koalaman](https://github.com/koalaman) commented at [2018-06-11T03:27:28Z](https://github.com/koalaman/shellcheck/issues/1242#issuecomment-396112787):

This is a very strange pattern, but still a valid issue. The simpler and more portable way of doing this is `printf '!!! Too many errors !!!\n'`

This happened because ShellCheck didn't correctly see `$'..'` as a literal string, but now it does. Thanks!

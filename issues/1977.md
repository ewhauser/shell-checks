#### <img src="https://avatars.githubusercontent.com/u/243351?v=4" width="50">[brother](https://github.com/brother) commented at [2020-06-04 06:27](https://github.com/koalaman/shellcheck/issues/1977#issuecomment-638630424):

Maybe use `checkbashisms` instead?

```
brother ~$ cat /tmp/test.sh 
#!/usr/bin/sh
var="foo_BAR"
echo "${var,} ${var,,} ${var^} ${var^^}"
brother ~$ shellcheck /tmp/test.sh
brother ~$ checkbashisms /tmp/test.sh
possible bashism in /tmp/test.sh line 3 (${parm,[,][pat]} or ${parm^[^][pat]}):
echo "${var,} ${var,,} ${var^} ${var^^}"
```

(checkbashisms is available in the devscripts package on debian based systems. current VCS location seems to be [salsa](https://salsa.debian.org/debian/devscripts/-/blob/master/scripts/checkbashisms.pl))

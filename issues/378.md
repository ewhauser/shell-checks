#### <img src="https://avatars.githubusercontent.com/u/2818758?v=4" width="50">[koalaman](https://github.com/koalaman) commented at [2015-06-14T16:33:35Z](https://github.com/koalaman/shellcheck/issues/378#issuecomment-111847392):

Apparently in `cmd.exe` and `powershell.exe`:
1. Make sure `chcp` says `65001` (UTF-8). This can be set with `chcp 65001`
2. Make sure to use a TrueType font, not a Raster Font, in Properties->Font.

In Powershell ISE you may additionally have to:
3. `[Console]::OutputEncoding = [System.Text.Encoding]::UTF8`
because chcp doesn't change it automatically.

Does that help?

# [\#406 Issue](https://github.com/koalaman/shellcheck/issues/406) `closed`: SC2086 exception: Wild card inside double quote

#### <img src="https://avatars.githubusercontent.com/u/495621?v=4" width="50">[baxeno](https://github.com/baxeno) opened issue at [2015-07-02T15:02:43Z](https://github.com/koalaman/shellcheck/issues/406):

When fixing SC2086 issues there is an exception when the string contain a wild card.

**Problem code - Remove multiple files:**

```
rm ${DATABASE_STORAGE}/NetworkList*
   ^-- SC2086: Double quote to prevent globbing and word splitting.
```

**Solution with pitfall:**

``` bash
rm "${DATABASE_STORAGE}/NetworkList*"
```

**Solution without pitfall:**

``` bash
 find "${DATABASE_STORAGE}" -name "NetworkList*" -print0 | xargs -0 -r rm -f
```

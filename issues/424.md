#### <img src="https://avatars.githubusercontent.com/u/2398949?v=4" width="50">[rony36](https://github.com/rony36) opened issue at [2015-08-04T09:08:05Z](https://github.com/koalaman/shellcheck/issues/424):

"local" can only be used in a function in Bash.
It will causes an execution error.
Here is a simple example.

``` bash
#!/bin/bash

local word="hi"
echo $word
exit 0
```

**line 3: local: can only be used in a function**

May shellcheck check for this error?

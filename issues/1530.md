# [\#1530 Issue](https://github.com/koalaman/shellcheck/issues/1530) `open`: Use exit status variable immediately after the function call

#### <img src="https://avatars.githubusercontent.com/u/25425799?v=4" width="50">[Rajeswari-31](https://github.com/Rajeswari-31) opened issue at [2019-03-22T12:37:40Z](https://github.com/koalaman/shellcheck/issues/1530):

We are using ShellCheck for performing code review for Shell scripts. We need to add a check like exit status variable is used after every function calls in the script. Can anyone help to understand whether any related feature is already available in ShellCheck and need guidance to develop it if it is not available.

Code:

```
function1( ) {
    \\some code here
}
function1()
ls -lt
if ($?){
\\print executed successfully
}
```
Expected response:

Here, we are invoking function1() followed by ls -lt.
we want to ensure that this should not be the case and the if ($?) should be immediately written.


#### <img src="https://avatars.githubusercontent.com/u/7482557?v=4" width="50">[Gandalf-](https://github.com/Gandalf-) commented at [2019-03-25T01:38:58Z](https://github.com/koalaman/shellcheck/issues/1530#issuecomment-476026084):

Your example doesn't look like a shell script, but I think I understand what you're getting at. There isn't any feature like this already in Shellcheck; I'm not sure one makes sense. Not all functions have meaningful return values.

If you want to force exit code handling, you may want to use `set -e`. See `help set`

#### <img src="https://avatars.githubusercontent.com/u/2818758?v=4" width="50">[koalaman](https://github.com/koalaman) commented at [2019-03-29T01:33:17Z](https://github.com/koalaman/shellcheck/issues/1530#issuecomment-477831303):

As @Gandalf- says there's no "must use return value" type functionality.

There's also not yet a good way to map a `$?` back to its command to simplify implementing this, though this is a feature that has been requested before and I agree should exist, to allow warning about code like `mycommand; echo "Finished with $?"; if [ $? -eq 0 ]`

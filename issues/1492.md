#### <img src="https://avatars.githubusercontent.com/u/365456?v=4" width="50">[evulhotdog](https://github.com/evulhotdog) commented at [2021-10-31T16:33:12Z](https://github.com/koalaman/shellcheck/issues/1492#issuecomment-955745993):

Reviewdog supports it, if you want shellcheck w/ Github Annotations.

```
  shellcheck:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v1
      - name: shellcheck
        uses: reviewdog/action-shellcheck@v1
        with:
          github_token: ${{ secrets.github_token }}
          reporter: github-pr-review
          path: "." # Optional.
          pattern: "*.sh" # Optional.
          filter_mode: nofilter # Optional.
          fail_on_error: true
```

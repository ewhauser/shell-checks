# [\#3159 Issue](https://github.com/koalaman/shellcheck/issues/3159) `open`: SC2142 false positive with inline functions inside aliases

#### <img src="https://avatars.githubusercontent.com/u/102495?u=c7bd9ddf127785286fc939dd18cb02db0a453bce&v=4" width="50">[ssbarnea](https://github.com/ssbarnea) opened issue at [2025-03-14 09:20](https://github.com/koalaman/shellcheck/issues/3159):

We should not need to disable SC2142 in order to make use of inline functions, example:

```shell
# shellcheck disable=SC2142
alias mypy='f() { command mypy "${@:-.}"; }; f'
```

Code above adds `.` argument to mypy if is called without any arguments.

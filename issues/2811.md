#### <img src="https://avatars.githubusercontent.com/u/84648683?v=4" width="50">[wileyhy](https://github.com/wileyhy) commented at [2023-08-02 23:13](https://github.com/koalaman/shellcheck/issues/2811#issuecomment-1663084538):

The OP's example is a case where an absolute path contains whitespace.
Executing `$cmd` without double quotes would cause an error.

$ touch file
$ cmd='./bin - test/rm'
$ set -x ; $cmd file ; echo $? ; set -
+ ./bin - test/rm file
bash: ./bin: No such file or directory
+ echo 127
127

This is a reasonable example of how relying on word splitting actually is
very fragile.

However, bash can locate and execute commands in directories in PATH the
names of which contain whitespace, evidently (on Android, anyway).

$ pwd
/data/data/com.termux/files/home
$ mkdir './bin - test'
$ cp "$( command -v rm )" ./bin\ -\ test/
$ PATH="./bin - test:$PATH"
$ declare -p PATH                      declare -x PATH="./bin -
test:/data/data/com.termux/files/usr/bin:/data/data/com.termux/files/usr/bin/applets"
$ hash -r
$ hash
hash: hash table empty
$ hash -p "$( type -P rm )" rm
$ hash
hits    command
   0    ./bin - test/rm
$ touch foo
$ set -x ; rm foo ; echo $? ; set -
+ rm foo
+ echo 0
0
+ set -
$ hash
hits    command
   1    /data/data/com.termux/files/usr/bin/touch
   1    ./bin - test/rm

#### <img src="https://avatars.githubusercontent.com/u/2818758?u=40d36e49d938fda80abd46d6347b9bace1f73dd4&v=4" width="50">[koalaman](https://github.com/koalaman) commented at [2023-08-13 21:50](https://github.com/koalaman/shellcheck/issues/2811#issuecomment-1676471371):

You're right about the problem, of course.

ShellCheck intentionally excludes SC2086 from the first word in a command (with or without redirection). This is due to the empirical observation that such a warning is significantly more likely to break a script expanding a command+args, than it is to fix a script running an executable with spaces in the path.

I would prefer emitting warnings the way you describe it, but a concession was made to improve accuracy and reduce noise.

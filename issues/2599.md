#### <img src="https://avatars.githubusercontent.com/u/20175435?u=8749e0c57901f29f964fdb7b1aea648d1370983a&v=4" width="50">[emanuele6](https://github.com/emanuele6) commented at [2022-10-05 17:19](https://github.com/koalaman/shellcheck/issues/2599#issuecomment-1268714257):

It is not a false positive. Your code is not correct, and if you run it, you will get an error, because it is not correct: `bash: return: [[: numeric argument required`.

The correct way to write that is `[[ -f "$0" ]]; return`.

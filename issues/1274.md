#### <img src="https://avatars.githubusercontent.com/u/2818758?v=4" width="50">[koalaman](https://github.com/koalaman) commented at [2018-07-02T21:12:04Z](https://github.com/koalaman/shellcheck/issues/1274#issuecomment-401937889):

Distro packaging is independently handled by wonderful volunteers on each distro.

EPEL Package maintainer Greg Hellings informs me that new versions of ShellCheck are tricky because they'd have to use the RHEL7 GHC, which is the five year old 7.6.3. This compiler is far out of date, and is additionally blocked by ShellCheck due to a compiler optimizer bug (#1131). 

You can use a precompiled, statically linked binary instead (see https://github.com/koalaman/shellcheck/#installing) , or build your own version using Stack which will download an appropriate GHC version.

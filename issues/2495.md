#### <img src="https://avatars.githubusercontent.com/u/24446?u=03bc7e86949cf30a5ea6ab492b39be33e855ee12&v=4" width="50">[mrmuskrat](https://github.com/mrmuskrat) commented at [2023-10-03 15:32](https://github.com/koalaman/shellcheck/issues/2495#issuecomment-1745223372):

If you want to use a pre-commit hook without docker just use this:

```repos:
-   repo: local
    hooks:
    - id: shellcheck
      name: shellcheck
      description: Test shell scripts with shellcheck
      entry: shellcheck
      language: python
      types: [shell]
      require_serial: true # shellcheck can detect sourcing this way

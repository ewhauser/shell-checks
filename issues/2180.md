version: 0.7.1

```sh
    while read line; do
	# this stuff happens in the same shell as the main script
    done <<< "$(cat "$tempfile")"
```
This is a very handy way to iterate over the contents of a temp file, but have the iteration be in the same shell (not a subshell).

#### <img src="https://avatars.githubusercontent.com/u/20781397?u=5a1d9a1830d909e86123f6032537c303551360d8&v=4" width="50">[0x326](https://github.com/0x326) commented at [2021-03-20 17:49](https://github.com/koalaman/shellcheck/issues/2180#issuecomment-803433734):

The syntax of the snippet is incorrect. It should be:
```bash
    while read line; do
	# this stuff happens in the same shell as the main script
        :
    done <<< "$(cat "$tempfile")"
```

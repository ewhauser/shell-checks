#### <img src="https://avatars.githubusercontent.com/u/167893?v=4" width="50">[mcarpenter](https://github.com/mcarpenter) commented at [2013-10-22T10:18:51Z](https://github.com/koalaman/shellcheck/issues/6#issuecomment-26791520):

I got it: I tried in a fresh Ubuntu 12.04 VM with just ghc installed on a fresh clone:

```
cabal update
cabal install
```

This worked perfectly and does not depend on Ubuntu's packaging. Closed! Thanks.

Similarly removing the following packages solved the issue on my original machine:

```
libghc-json-dev
libghc-parsec2-dev
libghc-parsec3-dev
libghc6-parsec3-dev
libghc-syb-dev
libghc6-json-dev
libghc-text-dev
```

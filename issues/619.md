#### <img src="https://avatars.githubusercontent.com/u/2818758?v=4" width="50">[koalaman](https://github.com/koalaman) commented at [2016-03-02T00:44:40Z](https://github.com/koalaman/shellcheck/issues/619#issuecomment-190994420):

Shellcheck is right, even if not especially pedagogic:

```
 SC1019: Expected this to be an argument to the unary condition.
```

What you have is equivalent to `[ -n "-t" ]` which is always true. To check if stdin is a terminal, use `[ -t 0 ]`.

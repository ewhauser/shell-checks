# [\#167 Issue](https://github.com/koalaman/shellcheck/issues/167) `closed`: Incorrect if...then

#### <img src="https://avatars.githubusercontent.com/u/32782?v=4" width="50">[michaelward82](https://github.com/michaelward82) opened issue at [2014-05-30T08:56:35Z](https://github.com/koalaman/shellcheck/issues/167):

I was suffering problems with a script because of a simple and easy to miss syntax error:

I had an if statement setup as follows:

```
if [ condition ];
then
    ...
fi
```

Of course, either the semi-colon should be omitted and `then` placed on a new line, or `then` should be moved up to the same line as the condition.

Would it be possible to add a check for this?

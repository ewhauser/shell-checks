I am trying to build shellcheck, but it fails with:

$> cabal build

fails, please run configure manually.
Warning: ShellCheck.cabal: Ignoring unknown section type: custom-setup

directory >=1.2.3.0

I am new to Haskell, and wouldn't know how to locate the rather generically named "directory" package. I tried google, which has an overabundance of results, and tried the sources in my package manager, and installed all that might remotely pertain to 'directory', but it still fails.

Any ideas?


#### <img src="https://avatars.githubusercontent.com/u/2818758?v=4" width="50">[koalaman](https://github.com/koalaman) commented at [2019-08-01T16:36:46Z](https://github.com/koalaman/shellcheck/issues/1658#issuecomment-517364664):

You can use `cabal update` followed by `cabal install --dependencies-only` to download and build all dependencies of the current project. Afterwards, `cabal build` and similar should work fine.

Please reopen if that doesn't fix it!

#### <img src="https://avatars.githubusercontent.com/u/2818758?v=4" width="50">[koalaman](https://github.com/koalaman) commented at [2013-11-18T18:34:26Z](https://github.com/koalaman/shellcheck/issues/14#issuecomment-28724573):

```
shellcheck --exclude=SC2086,SC2035 file
```

There's no config file yet, but you can make it somewhat more persistent by adding an alias in `.bashrc`:

```
alias shellcheck='shellcheck --exclude=SC2086'
```

#### <img src="https://avatars.githubusercontent.com/u/302055?v=4" width="50">[deryni](https://github.com/deryni) commented at [2015-08-31T00:18:31Z](https://github.com/koalaman/shellcheck/issues/448#issuecomment-136220204):

The only potentially meaningful difference between the two, that I can think of, is the difference in timing of when the file is opened (and how that might change race conditions). There may also be differences in how grep can operate on the input when it comes from standard input versus when it comes from an opened file (though grep might be smart enough to figure that out and work around it). I don't think either of those is material enough to warrant a warning/comment/etc. in either direction though.

There are many library scripts, which are never executed directly and are only sourced. Shebang mostly makes sense if the script has the executable flag.

#### <img src="https://avatars.githubusercontent.com/u/2818758?v=4" width="50">[koalaman](https://github.com/koalaman) commented at [2015-02-23T02:29:04Z](https://github.com/koalaman/shellcheck/issues/317#issuecomment-75481514):

Shellcheck needs to know the type of file it's checking. You can specify the shell with e.g. `-s bash` if you don't want to use a shebang.

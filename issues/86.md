# [\#86 Issue](https://github.com/koalaman/shellcheck/issues/86) `closed`: Look inside traps for used variables

#### <img src="https://avatars.githubusercontent.com/u/1326261?v=4" width="50">[MartinMSPedersen](https://github.com/MartinMSPedersen) opened issue at [2014-02-03T13:29:17Z](https://github.com/koalaman/shellcheck/issues/86):

This code gives a warning about LOOP_PID might be unused.

```
#!/bin/bash

trap 'kill $LOOP_PID' 0
eval "$(resize)"
while true
do
    for ((i=1; i<COLUMNS;i++))
    do
        echo -n '.'
        sleep 0.05
    done
    for ((i=1; i<COLUMNS;i++))
    do
        echo -en ' \b\b'
        sleep 0.05
    done
done &
LOOP_PID=$!
sleep 30
```

^-- SC2034: LOOP_PID appears unused. Verify it or export it.

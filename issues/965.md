#### <img src="https://avatars.githubusercontent.com/u/2199511?v=4" width="50">[Nightfirecat](https://github.com/Nightfirecat) commented at [2017-08-15T17:14:22Z](https://github.com/koalaman/shellcheck/issues/965#issuecomment-322529275):

Given that [there's an issue for another missing wiki page](https://github.com/koalaman/shellcheck/issues/969), I figured I'd do a bit of investigation in this. Below is a bash command that lists all errors/warnings that have pages on the wiki:

```
curl -s https://github.com/koalaman/shellcheck/wiki | grep -E 'wiki/SC[[:digit:]]{4}" class="wiki-page-link"' | sed -r -e 's/.+>(SC[[:digit:]]{4})<.+/\1/'
```

I'm sure a similar command could search the repo for instances of `/sc[\d]+/i` and remove duplicates.

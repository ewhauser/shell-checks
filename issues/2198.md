#### <img src="https://avatars.githubusercontent.com/u/2818758?u=40d36e49d938fda80abd46d6347b9bace1f73dd4&v=4" width="50">[koalaman](https://github.com/koalaman) commented at [2021-04-06 17:03](https://github.com/koalaman/shellcheck/issues/2198#issuecomment-814281959):

The warning is indeed confusing, but it does indicate a real problem. You should not use `[[ $(grep -q ..) ]]` because it has no output, and therefore will always be false. You should remove the brackets:

```
if [[ "$lsb_dist" == "rhel" ]] && ! grep -q "VERSION_ID=\"7" /etc/os-release; then
	sudo sed -i 's/$releasever/7/' /etc/yum.repos.d/docker-ce.repo
fi
```

This might deserve its own warning

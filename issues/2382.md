Everyone has its own difference.
Mine has these characteristics:
- Very fast, it autodetect shell files also the ones with no extension or wrong extension, and on my repo it take just 6 seconds to run, excluding checkout
- It doesn't use dockerfile
- It support ubuntu / macos / windows runners
- It has readable and coloured output and it is still able to use a matcher to report all errors / warnings /etc.

And most importantly it can automatically use the latest version of ShellCheck, in the second it get released (no commit needed).

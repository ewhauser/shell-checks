#### <img src="https://avatars.githubusercontent.com/u/16302148?v=4" width="50">[SimonHeimberg](https://github.com/SimonHeimberg) commented at [2020-06-30T07:36:08Z](https://github.com/koalaman/shellcheck/issues/1764#issuecomment-651608152):

Sourcing a bash file (b.sh) from a sh file (d.sh) will often result in problems (above: `bad substitution`).

The other way around (sourcing a sh file from a bash file) is no problem. (There may be some edge cases.)

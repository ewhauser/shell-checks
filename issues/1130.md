#### Here's a snippet that shows the problem:

```sh

#!/bin/sh
foo () {
	echo foo
}
export -f foo
```

#### Here's what shellcheck currently says:

```
$ sh -n foo.sh
$ sh foo.sh
foo.sh: 5: export: Illegal option -f
$ shellcheck foo.sh
$ checkbashisms foo.sh
possible bashism in foo.sh line 5 (export only takes -p as an option):
export -f foo
$ bash -n foo.sh
$ bash foo.sh 
$ 
```

#### Here's what I wanted or expected to see:

Exporting functions is not portable to shells other than bash.

# [\#3184 Issue](https://github.com/koalaman/shellcheck/issues/3184) `open`: Stuck in a loop

#### <img src="https://avatars.githubusercontent.com/u/76716?v=4" width="50">[grahamc](https://github.com/grahamc) opened issue at [2025-04-06 20:17](https://github.com/koalaman/shellcheck/issues/3184):

I have a file called `build-dry.sh`, and it looks like this:

```sh
source common.sh

export NIX_TESTS_CA_BY_DEFAULT=1

cd .. && source build-dry.sh
```

When I run shellcheck on it, it blocks forever. I believe it is recursively checking itself? Is there a way to ask it to not follow that source?

#### <img src="https://avatars.githubusercontent.com/u/76716?v=4" width="50">[grahamc](https://github.com/grahamc) commented at [2025-04-06 20:20](https://github.com/koalaman/shellcheck/issues/3184#issuecomment-2781633798):

...I worked around it like this:

```
# shellcheck disable=SC1091 source=/..
cd .. && source build-dry.sh
```

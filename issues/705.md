## Issue

SC2154 appears to be flagged in associative arrays when certain GNU tool names (e.g. tar, grep, sed, etc.) are used as keys.
## Environment

**OS**: macOS 10.11.5
**Shellcheck version**: 0.4.4
## Reproduction Steps

``` sh
# Inside test.sh
#!/usr/bin/env bash

declare -A flagged_packages
flagged_packages[grep]=--with-default-names
flagged_packages[gnu-sed]=--with-default-names
flagged_packages[gnu-tar]=--with-default-names
```
## Testing

``` sh
shellcheck test.sh
```

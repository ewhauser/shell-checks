# [\#3352 Issue](https://github.com/koalaman/shellcheck/issues/3352) `open`: SC2129 - If there is already a grouped redirection, SC2129 cannot be triggered

#### <img src="https://avatars.githubusercontent.com/u/103735539?v=4" width="50">[vivodi](https://github.com/vivodi) opened issue at [2025-11-09 04:12](https://github.com/koalaman/shellcheck/issues/3352):

The following *problematic* code does **not** trigger [SC2129](https://www.shellcheck.net/wiki/SC2129) — if there is already a grouped redirection, [SC2129](https://www.shellcheck.net/wiki/SC2129) cannot be triggered:

```bash
{
  echo foo
  date
} >> file
cat stuff >> file
```

Correct code:
```bash
{
  echo foo
  date
  cat stuff
} >> file
```

#### <img src="https://avatars.githubusercontent.com/u/2818758?u=40d36e49d938fda80abd46d6347b9bace1f73dd4&v=4" width="50">[koalaman](https://github.com/koalaman) commented at [2024-10-27 20:09](https://github.com/koalaman/shellcheck/issues/3071#issuecomment-2440158979):

You get this when you have 3 lines with the same redirects in a row. The example [from the wiki](https://www.shellcheck.net/wiki/SC2129) is:

```
echo foo >> file
date >> file
cat stuff  >> file
```

vs

```
{
  echo foo
  date
  cat stuff
} >> file
```

If you don't consider this an improvement, you can ignore the suggestion. If it's triggering where you didn't expect it, you can post a complete example.

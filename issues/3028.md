#### <img src="https://avatars.githubusercontent.com/u/16061266?v=4" width="50">[josephcsible](https://github.com/josephcsible) commented at [2024-07-26 03:42](https://github.com/koalaman/shellcheck/issues/3028#issuecomment-2251902635):

It's specifically Amazon Linux 2023 where it's unavailable. It's still available in DNF on basically every other Fedora-based distro. Details follow:

ShellCheck is still available in Fedora's repos, and in EPEL for RHEL-based distros. Amazon Linux 2 was roughly a clone of RHEL 7, and ShellCheck there came from EPEL, which itself came from Amazon-Linux-Extras. Unfortunately, Amazon Linux 2023 doesn't support that anymore: see https://repost.aws/knowledge-center/ec2-enable-epel and https://aws.amazon.com/linux/amazon-linux-2023/faqs/ for more details.

If you really want it from DNF, then switch to Fedora, RHEL, or one of RHEL's clones. If you're stuck on Amazon Linux 2023, then see https://github.com/koalaman/shellcheck/blob/master/README.md#installing for some non-DNF ways you can install it.
